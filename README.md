# notion_api_python
#### Notion API를 이용하여 데이터를 가져오는 방법을 정리한 문서입니다.
#### 데이터를 가져올 때 쓰레드를 이용하여 데이터를 가져오는 방법을 정리하였습니다.



### API Key 참조 
    # 'has_more': True, -> 다음 페이지가 있는지 확인
    # 'next_cursor': '4e45b1f8-b357-4bf7-ae94-55e4da52985d', -> start_cursor로 입력하면  다음 페이지가 출력된다

    # 'has_more': False, -> 다음 페이지가 없음
    
### 
    # 'object': 'list', -> 리스트 형식으로 출력
